# Functional Interfaces

- Exists since Java 1.0

**Definition**

- A Functional Interface (SAM) is an interface that has exactly one abstract method

```
@FunctionalInterface
```

- This annotation is introduced as part of the JDK 1.8
- Optional annotation to signify an inteface as Functional Interface

---

### New Functional Interfaces in Java8

- Consumer
  - BiConsumer
- Predicate
  - BiPredicate
- Function
  - BiFunction, UnaryOperator, Binary Operator
- Supplier
